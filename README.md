# Vue-order

基于vue.js+node.js+mongodb开发的订餐系统，代码注释详细，适合初学者.


PS：并不是所有页面都是用Vue.js做前台数据绑定，只是首页使用了Vue.js做数据的展示；
其他页面因Node.js方便而用了ejs渲染，样式使用Boostrap3。

<a href="http://vueorder.duapp.com/" target="_blank">http://vueorder.duapp.com/</a>
